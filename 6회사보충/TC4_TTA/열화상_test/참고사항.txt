열화상 온도 탐지

-----사용 데이터셋
1) 전체 데이터셋(유형 : csv / 수량 5개)
- 전체 데이터셋 : 한 파일당 2일(1분 간격) 길이의 데이터, 2880개의 행으로 구성됨

2) 테스트 데이터셋
- 테스트 구간 : 한 파일당 1시간 단위의 10개 구간 지정

3) 테스트 range(= 학습구간, = 입력값)
- 한 구간(1시간 단위)당 1분(1회씩)마다 sliding 방식으로 진행, 연속 5회
- 1구간 : 60개 * 5회 = 300개
- 2구간 : 60개 * 5회 = 300개
-
-
- 총 10개 구간 : 300개 * 10회 = 3000개

4) 예측값(출력값)
- 한 구간(1시간 단위)당 30분 단위의 예측값 출력
- 1구간 : 30개 * 5회 = 150개
- 2구간 : 30개 * 5회 = 150개
-
-
-총 10개 구간 : 150개 * 10회 = 1500개


-----시험 순서
1. 필요함수 불러오기
2. 데이터 불러오기 및 확인
3. 각 시료별 Test구간 설정 및 확인
4. Test기간 테스트 후 알람확인
5. 결과확인(csv)
3~5번 반복시행(각 시료별)
